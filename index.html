fragment.js

A minimalistic (0.4kB gzipped) tool for easily loading html fragments.

Example

To use fragment.js is as easy as setting the data-fragment attribute. 

        <div data-fragment="fragment.html"></div>
      


      

      
Templating example

      

        If either Mustache, Handlebars or Underscore
      

      
        <div data-fragment="mustache-fragment.html"
             data-fragment-json="fragment.json"></div>
